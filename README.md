# Projekt: Książka Adresowa (jQuery)

### Opis projektu
Aplikacja webowa służąca do zarządzania kontaktami. Pozwala na dynamiczne dodawanie wpisów, ich walidację oraz zaawansowane filtrowanie danych.

### Wykorzystane technologie:
* **jQuery** (Manipulacja DOM, animacje, obsługa zdarzeń)
* **jQuery Validate** (Walidacja formularza po stronie klienta)
* **DataTables** (Sortowanie, wyszukiwanie i paginacja tabeli)
* **Bootstrap 5** (Responsywny layout)

### Funkcjonalności:
* **Dodawanie kontaktów:** Formularz z walidacją pól (wymagane pola, format e-mail, długość telefonu).
* **Tabela dynamiczna:** Każdy kontakt trafia do tabeli obsługującej paginację i wyszukiwanie.
* **Animacje:** Efekt `fadeIn` przy dodawaniu kontaktu oraz `fadeOut` przy usuwaniu.
* **Interakcja:** Przyciski zmieniają skalę po najechaniu kursorem (`hover`).
